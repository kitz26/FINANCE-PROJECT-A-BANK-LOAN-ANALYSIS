# Bank Loan Finance Analysis (SQL Project)

A beginner-to-intermediate SQL project analyzing bank loan customer data — 
covering loan approvals, credit scores, income levels, and city-wise trends.

## Dataset
10 sample bank loan customers with fields like Loan Amount, Interest Rate, 
Loan Term, Monthly Income, Credit Score, and Loan Status.

## Questions Answered
- Total customers and total loan amount
- Average interest rate
- Approved vs rejected loans
- Highest loan amount
- Average loan amount by city
- Customers with credit score above 750
- Top 5 highest income customers
- Average income of approved customers

## Skills Used
SELECT, WHERE, GROUP BY, ORDER BY, COUNT(), SUM(), AVG(), LIMIT, 
filtering, and aggregate functions.

## Files
- `bank_loan_analysis.sql` — table creation, data insertion, and all queries
